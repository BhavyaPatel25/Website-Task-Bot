- Configurable via environment variables or a config file
- Easy to extend with new tasks or integrations

Quick start
1. Clone the repo:
   git clone https://github.com/BhavyaPatel25/Website-Task-Bot.git
2. Install dependencies:
   pip install -r requirements.txt
3. Run the bot (replace `bot.py` with the actual entry script if different):
   python bot.py
